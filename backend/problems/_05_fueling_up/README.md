## Part One

The teleporter requires exactly X pounds of fruit to initiate a jump.

Each fruit has:

- a weight (in pounds)
- an energy value

When inserted into the teleporter, each fruit contributes its full energy value. Fortunately, the system seems to have stabilized—there’s no strange decay or ordering issues to worry about. Each fruit can be used at most once.

Using the available fruits, determine the maximum total energy you can generate while using exactly X pounds of fruit.

## Part Two

Just as you prepare to activate the teleporter, the terminal crackles again:

“Warning: Fuel replication protocol enabled.”

You watch in amazement as a fruit you insert is… still there.

The system appears to have a duplication glitch:

You may now use each type of fruit any number of times.

However, this introduces a new constraint. The teleporter begins flashing:

“Minimize fuel instability. Reduce item count.”

It seems that while you can use as many fruits as you like, using too many may destabilize the jump.

What is the minimum number of fruits required to reach exactly X pounds? You may use each fruit unlimited times
